The ideal plugin to check your last login ip and time for his account security. Extra details like browser and operating system.



== Description ==

A plugin for admin use, which allows users to check their last login activity in their dashboard. 

No settings required, a Widget wrap with title "Last Login Details" will be displayed on your dashboard. Simply activate and check the last login record from your next logins. No extra scripts, no heavy load on database. No other activity trackings are included.



== Installation ==

Installation is free, quick, and easy. just upload to your plugins directory and activate.


== Frequently Asked Questions ==

= is it heavy on database? =
No it is not. it just saves all data in one field per user. So no extra chowking of data.


= Can I see the logins details of other users? =

No, you can see last login of you only.
